## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

- Update personal information in `app/page.tsx`
- Add your project images to `public/images/`
- Modify project data in the `projects` array
- Customize colors and styling in `app/globals.css`

## Tech Stack

- **Framework:** Next.js 15
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Motion library
- **UI Components:** Radix UI
- **Icons:** Lucide React

## Deployment

This portfolio can be easily deployed to Vercel, Netlify, or any other hosting platform that supports Next.js.
